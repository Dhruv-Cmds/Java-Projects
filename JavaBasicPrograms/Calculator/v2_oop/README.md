Calculator v2 (OOP)

A simple console-based calculator built in Java using Object-Oriented Programming principles.
Each mathematical operation is implemented in a separate class for clear structure and maintainability.

📌 Features

Addition (+)

Subtraction (-)

Multiplication (*)

Division (/)

Power (^)

🧱 Project Structure

Calculator.v2_oop
│
├── ContactManager.v1_oop.Main.java
├── Addition.java
├── Subtraction.java
├── Multiplication.java
├── Division.java
└── Square.java

▶️ How It Works

User enters:

First number

Operator (+, -, *, /, ^)

Second number

The program selects the corresponding class.

The operation is executed.

The result is displayed.
