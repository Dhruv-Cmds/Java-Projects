🏦 Java Console Banking App

- A simple console-based banking system built with Java.
- Straightforward. Functional. No nonsense.

- ✨ What It Does

- 💰 Check account balance

- ➕ Deposit money

- ➖ Withdraw money

- 🚪 Exit safely

🛠 Tech Stack

- Java

- Scanner (user input)

- Loops & Switch expressions

▶️ Run It
- javac Bankig/ContactManager.v1_oop.Main.java
- java Bankig.ContactManager.v1_oop.Main

🔒 Smart Validation

- Prevents negative deposits

- Blocks overdrawing

- Handles invalid menu selections
