# WeightConverstionProgram

## Description

A Java program that converts weight values between different units such as kilograms and pounds.

---

## Features

- Console-based Java application
- Clean and beginner-friendly logic
- Demonstrates core Java fundamentals
- Structured using packages

---

## Technologies Used

- Java
- OOP Concepts
- Console Input/Output

---

## How to Compile and Run

Make sure you are in the project root directory.

### Compile
```
javac WeightConverstionProgram\Main.java
```

### Run
```
java WeightConverstionProgram.Main
```
