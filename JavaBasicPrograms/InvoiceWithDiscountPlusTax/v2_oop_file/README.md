Invoice With Discount Plus Tax (OOP)

A simple Java console application that calculates an invoice total by applying discount and tax rules using Object-Oriented Programming.

Features

Calculates original amount

Applies discount or tax based on total

Computes final payable amount

Clean separation of logic using classes
