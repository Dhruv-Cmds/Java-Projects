
Student Grade System
Description

A simple Java console application that manages student grades using arrays.
The program collects student names and grades, then calculates and displays key statistics.

Features

Input number of students

Store student names in an array

Store student grades in an array

Display:

Highest grade with student name

Class average

Number of students who passed (grade ≥ 50)

Technologies Used

Java

Scanner class

Arrays

Loops and conditionals

How to Run

Compile the program:

javac StudentGradeSystem.java

Run the program:

java StudentGradeSystem
