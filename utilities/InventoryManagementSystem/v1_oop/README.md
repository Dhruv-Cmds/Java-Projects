# Inventory Management System 

A simple console-based Inventory Management System built in Java.
This project was created to practice core Java programming concepts such as classes, objects, arrays, loops, and menu-driven applications.

## Features

* Add new products to inventory
* View all products
* Search for a product by ID
* Update product quantity
* Calculate total inventory value
* Simple console menu for interaction

## Concepts Practiced

* Java Classes and Objects
* Constructors
* Arrays of Objects
* Loops and Conditional Statements
* Method Design
* Console Input using Scanner
* Basic Program Structure

## Project Structure

```
InventoryManagementSystem
│
├── Product.java
├── InventoryManager.java
└── Main.java
```

### File Descriptions

**Product.java**
Represents a product with fields such as:

* id
* name
* price
* quantity

**InventoryManager.java**
Handles inventory operations like:

* adding products
* searching products
* updating quantity
* calculating total value

**MainApp.java**
Contains the main method and menu-driven interface for interacting with the system.

## How to Run

1. Clone or download the repository.
2. Open the project in IntelliJ IDEA.
3. Navigate to `MainApp.java`.
4. Run the program.
5. Use the console menu to interact with the system.
