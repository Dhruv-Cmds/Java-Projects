LifeManager

Overview

LifeManager is a console-based Java application that provides basic daily utility tools within a structured, menu-driven interface.
The project demonstrates fundamental Object-Oriented Programming concepts using multiple classes organized within a package.

Features:-

BMI Calculator
Study Time Analyzer
Random Motivation Generator
Birthday Countdown Timer
Continuous menu-driven execution with exit option
Project Structure

Package:
LifeManager.v2_oop

Classes:

Main – Controls application flow and menu system
UserInfo – Collects and stores user details
StudyTime – Evaluates weekly study performance
Bmi – Calculates Body Mass Index
BirthdayCounter – Displays countdown with delay
Motivation – Generates random motivational messages

Technologies Used:-

Java
OOP Principles (Inheritance, Encapsulation)
Scanner (User Input)
Random (Message Generation)
Thread.sleep (Countdown Effect)
How to Compile and Run
Ensure you are inside the project root directory.

Compile
javac LifeManager\v2_oop_file\Main.java
Run
java LifeManager.v2_oop.Main
