# 📞 Contact Manager (Java OOP)

A simple **Contact Manager console application** built with **Java and Object-Oriented Programming (OOP)**.

## 🚀 Features

* Add new contact
* Delete contact
* Search contact by name
* Show all contacts
* Input validation
* Menu-driven console interface

## 🧱 Project Structure

```
ContactManager
 └── v1_oop
      ├── Main.java
      ├── Contact.java
      └── ContactService.java
```

## 🧠 Concepts Used

* Java Classes & Objects
* ArrayList
* Constructor Injection
* Exception Handling
* Input Validation
* Switch Expressions

## ▶️ How to Run

1. Clone the repository
2. Open in IntelliJ IDEA
3. Run `Main.java`

